# APAN5310_Group7
ABC Foodmart  
This is the project submission for group 7 for Spring 2026  

## Project Components

- **DataFiles.zip**  
  Contains the CSV datasets used for the project.

- **foodmart_code.ipynb**  
  Jupyter Notebook implementing database schema creation (PostgreSQL via psycopg2) and the ETL pipeline for loading data (pandas + SQLAlchemy)  

- **Group7_ERD_project.png**  
  Entity Relationship Diagram (ERD) of the database schema created in pgAdmin4.

- **Metabase_dashboards.zip**  
  Exported PDF dashboards created in Metabase to visualize key metrics and insights.

- **Final_Report.pdf**  
  Comprehensive written report detailing methodology, analysis, and findings.

- **Presentation_Slides.pdf**  
  Final presentation summarizing project.  

## Tools & Technologies
- Python (Pandas, psycopg2, SQLAlchemy)
- PostgreSQL / pgAdmin4
- Metabase
- Jupyter Notebook

## How to Run
1. Unzip `DataFiles.zip` to access the datasets  
2. Open and run foodmart_code.ipynb:  
- Creates the PostgreSQL database schema  
- Loads data into the database via ETL pipeline  
3. Connect pgAdmin4 to PostgreSQL for SQL querying
4. Connect Metabase to PostgreSQL for dashboards
