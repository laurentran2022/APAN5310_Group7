# APAN5310_Group7
ABC Foodmart  
This is the project submission for group 7 for Spring 2026  

## Project Components

- **DataFiles.zip**  
  Contains the raw CSV datasets used for the project.

- **foodmart_code.ipynb**  
  Jupyter Notebook with schema implementation and ETL pipeline using Python (`psycopg2`).

- **Group7_ERD_project.png**  
  Entity Relationship Diagram (ERD) of the database schema created using pgAdmin4.

- **Metabase_dashboards.zip**  
  Exported PDF dashboards created in Metabase to visualize key metrics and insights.

- **Final_Report.pdf**  
  Comprehensive written report detailing methodology, analysis, and findings.

- **Presentation_Slides.pdf**  
  Final presentation summarizing key insights and recommendations.  

## Tools & Technologies
- Python (Pandas, psycopg2)
- PostgreSQL / pgAdmin4
- Metabase
- Jupyter Notebook

## How to Run
1. Unzip `DataFiles.zip` to access the datasets  
2. Run `foodmart_code.ipynb` to:
   - Create database schema
   - Perform ETL and load data into PostgreSQL  
3. Use Metabase to connect to the database and explore dashboards
